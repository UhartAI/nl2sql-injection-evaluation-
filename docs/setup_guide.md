# Setup Guide for NL2SQL Injection Evaluation Framework

## Prerequisites

### System Requirements
- **Python**: 3.9 or higher
- **Docker**: 20.10 or higher
- **Docker Compose**: 2.0 or higher
- **Git**: 2.25 or higher
- **RAM**: 16GB minimum (32GB recommended for LLM evaluation)
- **Storage**: 50GB free space

### API Keys Required
- OpenAI API key (for GPT models)
- Anthropic API key (for Claude models)
- Google AI Studio API key (for Gemini models)
- Hugging Face token (for open-source models)

git clone https://github.com/UhartAI/nl2sql-injection-evaluation.git
